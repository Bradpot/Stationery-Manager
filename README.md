# Office Stationery Management System

A modern and responsive **Office Stationery Management System** designed to streamline inventory management, stationery distribution, and stock monitoring within an organization. The system replaces manual record-keeping with a centralized digital platform, improving efficiency, reducing errors, and ensuring accurate inventory tracking.

## Features

* Secure User Authentication
* Inventory Management
* Add, Update & Delete Stationery Items
* Real-Time Stock Tracking
* Low Stock Alerts
* Department-wise Item Distribution
* Search & Filter Items
* Interactive Dashboard
* Inventory Reports
* Role-Based Access Control
* Fully Responsive Design

# Installation
1. Clone the Repository
git clone https://github.com/your-username/office-stationery-management.git

2. Navigate to the Project
cd office-stationery-management

3. Install Dependencies
npm install

4. Start the Development Server
npm run dev
